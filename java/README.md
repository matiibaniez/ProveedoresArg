# ProveedoresArg – Prototipo POO en Java (AP3)

Este módulo implementa un prototipo funcional del sistema aplicando los pilares de la Programación Orientada a Objetos:
- Encapsulamiento
- Herencia
- Polimorfismo
- Abstracción
